Este programa funciona abriendo una txt. y tomando los datos en él formando una matriz 
con m x n dimensiones y posteriormente aplica el algoritmo de eliminación Gaussiana que primero
verifica que la primera fila no empiece con 0, para luego multiplicar la primera fila en un número que sumado a la segunda fila quede 
cero debajo del primer numero, y posteriormente seguir con la segunda fila hasta terminar las n filas.
Despues de terminar las filas, pasa a la siguiente columna y realiza lo mismo hasta que haya cero debajo de cada primer elemento distinto
a cero de cada fila
